# Flow Icons for Zed

Vibrant and distinguishing icons for the Zed editor.

## Status: Pending Approval ⏳

This extension is currently pending approval to be included in the official Zed Extension Registry. In the meantime, you can install and use it as a **Dev Extension**.

## How to Install (Dev Extension)

1. **Clone this repository** to your local machine:
   ```bash
   git clone https://github.com/prsweet/vc-flow-icons-zed.git
   ```

2. **Open Zed**.

3. **Install as Dev Extension**:
   - Open the **Command Palette** (`Cmd+Shift+P` on macOS / `Ctrl+Shift+P` on Windows/Linux).
   - Search for and select: `extensions: install dev extension`.
   - Browse to and select the folder where you cloned this repository (`vc-flow-icons-zed`).

4. **Select the Icon Theme**:
   - Open the **Command Palette** again.
   - Search for `theme: select icon theme`.
   - Select **Flow Icons**.

---

Enjoy your vibrant new workspace icons! 🚀
